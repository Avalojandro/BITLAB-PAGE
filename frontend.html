Desarrollo web-Frontend
3 meses
Beca Disponible

En este curso, el estudiante aprenderá los fundamentos básicos del desarrollo web Frontend, para construir con éxito un producto web.
Al ser un curso teórico práctico el estudiante aplicará los conocimientos adquiridos los cuales le permitirán desarrollar un sitio web, capaces de consumir servicios Web y presentar datos al usuario de forma atractiva, bien estructurada y óptima; siguiendo los buenos principios de programación y utilizando tecnologías de vanguardia disponibles para esta plataforma.
